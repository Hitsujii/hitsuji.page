# Q1

The program is supposed to add two numbers, but doesn't work correctly.

Use the integrated debugger to step through the program and watch the value of `x`.

```cpp
#include <iostream>

int readNumber(int x)
{
	std::cout << "Please enter a number: ";
	std::cin >> x;
	return x;
}

void writeAnswer(int x)
{
	std::cout << "The sum is: " << x << '\n';
}

int main()
{
	int x {};
	readNumber(x);
	x = x + readNumber(x);
	writeAnswer(x);

	return 0;
}
```

## Solution

Debugger shows us the problem:

```bash
Breakpoint 3, main () at main.cpp:16
16          x = x + readNumber(x);
(gdb) print x
$1 = 0
(gdb)
```

`readNumber()` only returns a value, it does not change the original variable.

Assign the return value of the function to `x`:

```cpp
#include <iostream>

int readNumber(int x)
{
    std::cout << "Please enter a number: ";
    std::cin >> x;
    return x;
}

void writeAnswer(int x)
{
    std::cout << "The sum is: " << x << '\n';
}

int main()
{
    int x{};
	// readNumber(x);
    x = readNumber(x); // correct version
    x = x + readNumber(x);

    writeAnswer(x);

    return 0;
}
```

Now the result is correct:

```bash

Please enter a number: 3
The sum is: 5
cpp/sandbox/code took 2s
❯
```
