# Q2

The following program is supposed to divide two numbers, but doesn’t work correctly.

Use the integrated debugger to step through this program. For inputs, enter 8 and 4. Based on the information you learn, fix the following program:

```cpp
#include <iostream>

int readNumber()
{
	std::cout << "Please enter a number: ";
	int x {};
	std::cin >> x;
	return x;
}

void writeAnswer(int x)
{
	std::cout << "The quotient is: " << x << '\n';
}

int main()
{
	int x{ };
	int y{ };
	x = readNumber();
	x = readNumber();
	writeAnswer(x/y);

	return 0;
}
```

## Solution

`writeAnswer()` is dividing by `0`.

```cpp
writeAnswer(x / y); // y = 0
```

We never assign any value to `y`.
We only overwrite the value of `x`.

```cpp
x = readNumber();
x = readNumber();
```

Replace `x = readNumber();` with `y = readNumber();`:

```cpp
#include <iostream>

int readNumber()
{
    std::cout << "Please enter a number: ";
    int x{};
    std::cin >> x;
    return x;
}

void writeAnswer(int x)
{
    std::cout << "The quotient is: " << x << '\n';
}

int main()
{
    int x{};
    int y{};

    x = readNumber();
	// x = readNumber();
    y = readNumber(); // correct version

    writeAnswer(x / y);

    return 0;
}
```
