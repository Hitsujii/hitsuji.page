# Program flow

1. Define the goal.

```text
Read two numbers and print the result.
```

2. Write the requirements.

```text
The user enters two numbers.
The program calculates the result.
The program prints the result.
```

3. Break the problem into parts.

```text
- read first number
- read second number
- calculate result
- print result
```

4. Turn parts into functions.

```cpp
readNumber();
calculateResult();
printResult();
```

5. Decide the flow.

```text
input -> calculation -> output
```

6. Start with `main()`.

```cpp
int main()
{
    // readNumber();
    // calculateResult();
    // printResult();

    return 0;
}
```

7. Implement one piece at a time.

```text
prototype -> body -> test
```
